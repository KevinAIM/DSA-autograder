This project is meant to be an autograder that works with Canvas. It was created for a college level Data Structure and Algorithms (DSA) course. The user should edit the (file unmade) for their specific assignment. It will be sent to the LLM which will formulate a response after it automatically collects the student's file submission (.java) from Canvas. The response will be automatically send via Canvas as well. (Canvas integration not yet built)
